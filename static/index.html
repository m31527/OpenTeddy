    Webhook Endpoint
    外部系統可 POST 到 /webhooks/<session_id> 觸發該 session。 範例： 
    curl -X POST http://localhost:8000/webhooks/YOUR_SESSION_ID \
  -H "Content-Type: application/json" \
  -H "X-OpenTeddy-Webhook-Secret: YOUR_SECRET" \
  -d '{"goal": "check for new orders and telegram me if any"}'
